# Java Refresher

A Java refresher project based on a Udemy course covering modern Java development techniques from **Java 8 to 25**.

## About

This project was created as a beginner tutorial, but is also great for senior developers looking to refresh their skills — especially with the latest updates and changes introduced across modern Java versions.

## Course Overview

- Learn how to develop Java applications from the ground up
- Covers modern Java development techniques spanning Java 8 through Java 25
- Suitable for beginners and experienced developers alike

## Who Is This For?

- **Beginners** looking to learn Java development from scratch
- **Senior Developers** wanting to refresh their knowledge and catch up on recent Java changes

## Prerequisites

- Basic understanding of programming concepts (recommended for beginners)
- Java Development Kit (JDK) installed — Java 25 recommended

## Topics Covered

- Core Java fundamentals
- Object-Oriented Programming (OOP)
- Modern Java features (Streams, Lambdas, Records, Sealed Classes, and more)
- Updates and enhancements from Java 8 to Java 25
